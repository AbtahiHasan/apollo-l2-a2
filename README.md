## How to set up this project locally following few step

##### step 1 create .env file in root and this code into .env file

```bash
PORT=5000
DATABASE_URL=database url here
BCRYPT_SALT_ROUNDS=12
```

##### step 2 install

```bash
    npm i
```

##### step 3 run with ts-node-dev

```bash
    npm run dev
```
